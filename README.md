# Cinema Booking System

## Cómo usar la aplicación

1. Selecciona una película desde la cartelera principal.
2. Escoge la función y el horario disponible.
3. Selecciona los asientos deseados dentro de la sala interactiva.
4. Verifica el resumen de la reserva antes de continuar.
5. Confirma la selección para finalizar la simulación de compra.

---

## Recomendaciones de uso

- Los asientos ocupados no pueden seleccionarse.
- Puedes cambiar de función antes de confirmar la reserva.
- La disponibilidad de asientos se actualiza de forma interactiva.
- Verifica cuidadosamente la selección antes de confirmar.

---

## Navegación

- Usa el menú principal para volver a la cartelera.
- El panel de resumen muestra la información actual de la reserva.
- La interfaz está diseñada para funcionar tanto en escritorio como en dispositivos móviles.

---

## Nota

Esta aplicación es un simulador interactivo de reservas de cine con fines educativos y de práctica de desarrollo web.
